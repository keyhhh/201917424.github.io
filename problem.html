我对redis中锁的理解。

Redis/发现自学习秒杀模式

原理就是使用setnx key value ，因为setnx 方法创建key前会检查是否存在该key，否则不创建，因此视创建成功为上锁，可以进行数据操作，创建失败为其他线程已经上锁，无法进行操作； 解锁为del key，删除该key，同时为了防止阻塞，可以为key设置过期时间，到期自动解锁。

Linux虚拟机中如何与本地主机进行文件操作？

Linux/发现自需要向虚拟机上传软件安装包

方法一：再Linux中安装lrzsz，首先搜索lrzsz安装包：yum list lrzsz，再安装：yum install lrzsz.x86_64，需要上传文件时，控制台输入命令rz；方法二：下载安装Xshell ：是一个强大的安全终端模拟软件，开启VMWare打开虚拟机，使用Xshell连接虚拟机，可以实现文件互传、虚拟机控制台直接粘贴主机复制的内容等功能。

Linux的文件权限查看方式。

Linux/发现自使用ll命令

文件属性一共十位，第1位为文件类型:- 普通文件 ， d 目录文件，I 链接文件，p 管理文件， b 块设备文件， c 字符设备文件， s 套接字文件；2-4位：文件所属用户权限；5-7位：文件创建者所在用户组其他成员权限；8-10位：其他用户权限；权限所代表字母及数字：(r、可读、4)，(w、可写、2)，(x，可执行，1)，例如最高权限drwxrwxrwx 777;

Mysql主从复制操作出错。

Mysql/发现自项目练习使用两台虚拟机实现主从复制

我的第一种情况：删除主库的mysql-bin.0000001这类文件时，注意删除mysql-bin.index内容，否则从库无法无法对应上主库；第二种情况：直接手动用navicat删除主库内容后，从库没有删除并且后续操纵无法复制，将同步指针移动到下一指令就可以，跳过这个错误，首先停止同步，然后执行set global sql_slave_skip_counter = 1，最后在开启同步。
